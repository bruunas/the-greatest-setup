
# The Greatest Setup

**Requeriments**
- Node 10.13 ou maior
- Yarn


## Install & Start

- Install 
```bash
yarn
```

- Starts the development server.
```bash
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


- Builds the app for production.
```bash
yarn build
```

- Runs the built app in production mode.
```bash
yarn start
```
